# staySync 연결 도우미 — 개인정보처리방침

**최종 업데이트: 2026-06-09**

staySync 연결 도우미("본 확장")는 사용자가 운영하는 숙박 플랫폼의 예약 정보를
staySync 서비스로 안전하게 동기화하기 위한 Chrome 확장 프로그램입니다.
본 방침은 확장이 어떤 정보를 어떻게 처리하는지 설명합니다.

---

## 1. 수집하는 정보

본 확장은 **사용자가 직접 로그인한 플랫폼**에서 아래 정보만 처리합니다.

| 구분 | 내용 | 처리 방식 |
|------|------|-----------|
| 예약 정보 | 체크인/체크아웃 날짜, 예약 식별자, 게스트 표시명 | 사용자가 로그인한 화면(DOM)에서 읽어 staySync 서버로 전송 |
| 세션 토큰 | 플랫폼이 사용자 브라우저에 발급한 인증 토큰 | 일부 플랫폼 연동 시에만, 사용자 본인 staySync 계정에 연결 |

지원 플랫폼: 33m2(삼삼엠투), 자리톡, 엔코스테이, 리브애니웨어, 직방

## 2. 수집하지 않는 정보

본 확장은 다음을 **절대 수집·저장·전송하지 않습니다.**

- ❌ 플랫폼 또는 staySync **로그인 비밀번호**
- ❌ 결제수단·카드번호 등 금융정보
- ❌ 주민등록번호 등 민감 개인정보
- ❌ 사용자의 브라우징 기록 또는 다른 사이트 활동
- ❌ 지원 플랫폼 외 사이트의 어떠한 데이터

## 3. 정보 이용 목적

수집된 예약 정보는 오직 **사용자 본인의 staySync 계정에서
여러 플랫폼 예약을 통합 표시하고 이중예약을 감지**하는 목적으로만 사용됩니다.
제3자에게 판매·공유하지 않습니다.

## 4. 정보 전송 및 보관

- 예약 정보는 HTTPS로 staySync 서버(`api.staysync.kr`)에만 전송됩니다.
- 사용자 본인 staySync 계정에 귀속되어 보관되며, 계정 삭제 시 함께 삭제됩니다.
- 세션 토큰은 암호화하여 보관하며, 동기화 목적 외에는 사용하지 않습니다.

## 5. 사용자 권리

- 확장 팝업에서 언제든 플랫폼 **연결을 해제**할 수 있습니다.
- 확장을 삭제하면 신규 데이터 수집이 즉시 중단됩니다.
- staySync 계정 삭제를 요청하면 관련 데이터가 모두 삭제됩니다.

## 6. 권한 사용 사유

| 권한 | 사유 |
|------|------|
| `cookies` | 사용자가 로그인한 플랫폼의 세션 토큰을 읽어 연동 (비밀번호 아님) |
| `scripting` | 로그인된 예약 화면에서 예약 정보를 읽기 위해 |
| `storage` | 연결 상태를 로컬에 저장 |
| `tabs` | 현재 열린 플랫폼 탭 확인 |
| host_permissions(지원 플랫폼 도메인) | 해당 플랫폼 페이지에서만 동작하기 위해 |

본 확장은 **지원 플랫폼 도메인 및 staySync 도메인에서만** 동작하며,
그 외 사이트에는 어떠한 코드도 실행하지 않습니다.
